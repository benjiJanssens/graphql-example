# graphql-example

This is an example of GraphQL

## Installation

Use the package manager [npm](https://www.npmjs.com) or [Yarn](https://yarnpkg.com) to install the dependencies.

```bash
npm/yarn install
```

## Usage

### Start in development mode

```bash
npm run dev
```
